「汉化组定义」

汉化组之间由于沟通不畅等原因造成的效率损失，如「占坑不填」、「失联」、「撞车」等……

这样的事终究会给人挫败感，甚至会扑灭一些个人汉化者的热情。但正因为有这些贡献者，国外文化才得以走入大众视野。

为了保证这些工作者的劳动成果得到最大程度之尊重，我认为这些沟通困难、项目进度不十分透明、经验无法传承之类的问题有必要提供解决方案——即「汉化组数据库/汉化组布告栏」项目
